How Codex works

Codex understands your codebase, plans the work, executes changes, and validates results—all within a reliable loop you can trust.

1

Understand your codebase

Codex indexes your repository, builds a dependency graph, and understands your project structure, architecture, and conventions before making any change.

2

Plan the implementation

Given a task, Codex creates a detailed plan—identifying files to change, dependencies to update, tests to write, and edge cases to handle.

3

Execute with precision

Codex generates code, runs linters and type checkers, creates and updates tests, and iterates until all checks pass—producing production-ready changes.

4

Review and ship

Every change comes with a clear summary, test results, and diff preview. You review, approve, and merge—confident that the work has been thoroughly validated.

Frequently asked questions

Common questions about downloading and using Codex.

What is Codex?

Codex is an AI coding agent from OpenAI, powered by ChatGPT. It helps developers build and ship software faster by handling complex engineering tasks end to end—from feature development and refactoring to code review and testing. Codex is available as a desktop app for macOS and Windows, as IDE extensions, and as a CLI tool.

How do I download Codex?

Click the Download button at the top of this page to get the latest version of Codex for your platform. macOS users can download the DMG installer directly. Windows users can install Codex through the Microsoft Store. Both versions are free to download and use with your ChatGPT account.

Is Codex free to use?

Yes, Codex is free to download and use. It is powered by your ChatGPT account, and usage may be subject to your ChatGPT plan's limits. Teams can earn up to $500 in credits when their members start using Codex.

What operating systems does Codex support?

Codex is available on macOS and Windows. The macOS version is distributed as a DMG file, and the Windows version is available through the Microsoft Store. Codex CLI works on any platform that supports Node.js, including Linux.

Can Codex work with my existing codebase?

Yes. Codex indexes your repository, builds a dependency graph, and understands your project structure, architecture, and coding conventions. It works with any language and framework, and integrates with VS Code, JetBrains IDEs, and your terminal.

How does Codex compare to GitHub Copilot or Cursor?

Codex is designed for end-to-end engineering tasks rather than inline autocomplete. It handles multi-step workflows—planning, implementing, testing, and reviewing changes autonomously.
